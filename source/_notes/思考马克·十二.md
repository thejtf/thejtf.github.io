# **思考马克·十二**

### **2023年九月记录**



9月6日

谷东安老师留给这个世界的最后一句话：“我是来创造生活的，不是来委屈自己迁就这个时代的。”



9月10日

如果这是粪坑，你要做的不是清理它，而是要迅速远离。清理粪坑并不会给你成就感，只会让你一次次恶心。



9月12日

今天去公安局看门大爷说你得抵押点什么才能进去，我今天包里就只带了两本书，一本是黄仁宇的《万历十五年》英文版，一本是钱穆的《国史大纲》，最后我把后者给看门大爷了。大爷很惊奇说小伙子还看书呐？或许我应该是很少把书押给公安局门卫的人吧。进去后，等了很久办案民警，甚至《万历十五年》都看了好几页关于万历皇帝的事了。我其实很淡定，我早知道就是这种结果，因为我带的书里面都写尽了这样的故事。



9月13日

一些朋友听闻了我的近况给了我许多鼓励和安慰，我想我的人生真的是很精彩和幸运，我会记住这一切的，这些鼓励是我未来人生路上宝贵的盘缠。我没有你们说的那么好，但我确实也不坏，我妈给我算过命的，我的命运就是初限驳杂，但好消息是大器晚成。谢谢这么多人美好的祝愿，我想我命运的齿轮就要开始转动了。

\
9月14日

了解世界的黑暗与绝望，却从不放弃，并以悲天悯人之心去关怀所有不幸的人。这才是伟人之所以成为伟人的真正原因，这才是人类最为崇高的道德与情感。

\
9月15日

可知的世界可知，不可知的世界不可知，存在总是存在者的存在。或许真正的成功只有一个，就是按照自己喜欢的方式度过一生。我喜欢马斯克这段话，就像乔布斯说的，真正足以疯狂地相信自己可以改变世界的人才能改变世界。



9月16日

历史给我们的教训就是我们不会从历史中得到任何教训，以史为鉴？算了吧。技术在变，只要人性未变，变化的就只是表象，历史是车轮，它从未变过。以前犯的错，未来还会再犯。该杀的人，还是会再杀。伍子胥会死，韩信会死，岳飞会死，窦娥也依旧会蒙冤，再过一千年这类人的结局都不会改变。



9月18日

青年的爱国是有力量的，不像某些成年人的爱国到处充满了蝇营狗苟般的算计。



9月19日

我不是毛粉，但是我对他的理论极其感兴趣。一介书生，一个理想主义者是怎么一步步引导中国革命取得了胜利，我没资格评价他，但从他的文笔就能窥见一斑，会写文章的国人大有人在，会写文章又能搞革命的恐怕就只有他了。

\
9月20日

To err is human, to forgive devine.

\
9月21日

天不生仲尼，万古如长夜。



9月22日

一个方便的入口+一个强需求=一个好产品。

9月23日

拿着维特根斯坦是一点没看进去，打开豆瓣搜到鲁迅，再看他百年前的文字，五味杂陈。



9月25日

让人真正改变的从来不是知识而是见识。

\
9月26日

忽然发现，最近自己的判断力变得非常差。



9月28日

胡适先生说过，用自己的自由换取安稳，既得不到安稳也得不到自由。



9月29日

人生没有终点，但永远都会有下一站。



9月30日

一些历史或者一些书籍之所以看不懂是因为没法共情，伟大的作品需要理解它的人，厄运和苦难从来都是优秀作品的注解。
