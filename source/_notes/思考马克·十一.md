# 思考马克·十一

### 2023年八月记录



8月3日

一个人一天的心情是会变化的，从早晨的阳光到晚上的阴暗。



8月4日

有想法能实现那是幸福，有想法不能实现那就是烦恼。想法是烦恼也是菩提，烦恼即菩提。



8月5日

幸福三宝：别管、别听、别信。

\
8月6日

历史是真的，人是复杂的。我再也回不去那个用单一体系去评判对错的男孩了。知道事情的真相如何？了解到世界的复杂又如何？混乱是阶梯，前进是我们唯一能做的。

\
8月7日

有些鸟儿注定是关不住的，它们的每一片羽毛都闪耀着自由的光辉。一个人能成为谁，能否获得真正的自由，来自于你的内心，它可以超越很多东西。“你可以爱一个人，但仍然选择和他说再见。你也可以每天都想念一个人，但仍然庆幸他不在你的生命中”。



8月8日

人类似乎真的是硅基生物的安装器，我们存在的意义或许就是安装并启动硅基生物程序。



8月9日

你的产品有人用，还能愿意付费，这才是必备。



8月10日

一定要在到达山顶前考虑好接下来要爬的下一座山峰，否则下山的时候是容不得你再抉择什么的，只能是被动的下落。



8月11日

保护生态的意义似乎是“维持人类和诸生物的领土现状以及互不侵犯条约”，否则你把某昆虫动物的家平了，人家来袭击你，那还不是因果报应吗…



8月12日

越来越喜欢务实的人，做人得有目标感。



8月13日

《左传》真的好看，历史远比小说精彩，以史为镜当然可以知兴替。是由于不遵守周礼才使周王朝日渐甚微，也是一些诸侯打破了旧规则才成为一代枭雄，礼崩乐坏之际总是群雄豪杰人才辈出。总之要顺势而为，要借道行事。没有人改变了历史，一个时代总会走向终结，而我们仅仅都是历史微不足道的一部分。

8月14日

If justice is blind, she sometimes heads down unpredictable paths.



8月15日

太上有立德，其次有立功，其次有立言，传之久远，此之谓不朽。

\
8月16日

从来不会因为一个漏洞弃用一个产品，哪怕这个漏洞再严重。而肯定会因为一个团队对待漏洞的态度而决定弃用与否，哪怕这个漏洞再小。

\
8月17日

真的“极度渴望成功”的人其实并不多，“愿意付出非凡代价”的就更少了。



8月18日

做优质内容的前提是，你得有一批为优质内容的产出而付费的群体和一定的新闻信念与理想。这个世界的腐烂一定是从媒体堕落开始的，优质内容从来不是什么张嘴闭嘴的正能量。就像《基督教科学箴言报》和《卫报》的 slogan 一样，新闻可以是独立的、诚实的、充满希望的、可信的、有建设性的。就像春秋时代的史官一样如实的、客观的记录着这世间每天发生的历史。可新闻不再是新闻，历史不再是历史，我们每一个人都靠社交媒体汲取那讨我们喜欢的信息的时候，这个世界就早已注定腐烂了。



8月19日

历史会记录我们，后代会耻笑我们。记录我们的魔幻，耻笑我们的荒诞。我们把美好的明天断送，然后我们又把希望留给明天。就像1587年，天下盛平无重大动荡，是无关紧要的一年，但它也是大明王朝发展至尽头开始走向灭亡的一年，史称“万历十五年”。



8月20日

齐桓晋文，尊王攘夷。

禁抑篡弑，裁制兼并。



8月21日

从鲁隐公元年到鲁哀公十四年的春秋史，可以说是极其有趣。今天的我们看他们似乎有点傻，有点天真。但那个时候就是那么的看重道义礼信。或许今天的我们看他们总是有种距离的美感，但是不管怎样，他们都对我们后世之文化，影响深远。



8月24日

树立典型是最可怕的，无论是好的典型还是坏的典型。人是复杂的，好人也会有不好的品质，坏人也会有好的品质。可是当“某种典型”被树立之后，人们就会单一的对人去进行评价。这本身就是一种对立思维。



8月25日

不想把希望放在未来了，要把希望放在当下。

\
8月26日

我还是讨厌人群，因为我讨厌他们提出的傻逼问题

\
8月27日

评价一个产品好坏的重要一条就是：是否乱加臃肿的功能。



8月28日

人要有仁爱之心，有仁爱之心才能更好的自我实现。但它通常只是一个大前提，或者说是做好事的动机而已。人人都有仁爱之心但并没有让人类走向大同，只因背后真正博弈的并不是人的仁义而是人与人之间的利益。令人失望的是，“仁义”在历史上通常是种借口和手段，也就是理由正义。它通常是发动战争的结果，而不是发动战争的真正原因。正义永远都是一种能够调动人心的力量，人心士气占哪边，哪里就成功了一半。

8月29日

市场从来不是以你的流量大小来定价的，而是你的能影响到的用户的本身价值所决定的。



8月30肉

看《奥本海默》就记住一句台词：权力只存在阴影中。
