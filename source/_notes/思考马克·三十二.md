# **思考马克·三十二**

### **2025年九月记录**



8月21日

人想要不被淘汰，唯有一条路：一辈子不断OTA。OTA的当然不只有知识、还包括思维认知、偏好与个人性格。



8月24日

很多不善良都是来自公地悲剧，恶意一旦蔓延，人性是很难修正的。在一切人反对一切人的战争里，善意需靠思想引导，需要让渡自己的利益让群体利益最大化，需要靠人群的行为惯性积累。善良与宽容永远是群体生活的最优博弈策略，恶意虽然可以一时爽，但是生活的回旋镖迟早会飞向你。

\
8月25日

很多产品的支持者并不会发声，他们只是一味的默默用脚投票。产品最重要的用户可能并不会给 PM 任何反馈，而骂得最凶的、经常反馈的反而可能会是低价值用户，甚至是负价值用户。用户的真实购买、用钱投票才是唯一可靠的民意数据。提高胜率的重要手段是忽略一切主观评价，而把市场当作一个实验模型，收集尽可能多的客观数据，用严格的数学方法提取客观可靠的变量间的数学关联才是最行之有效的办法。

\
8月26日

地里已经没有果实了，一些人不想着怎么种地，都在想着怎么收割……



8月27日

在文字和思想面前众生是平等的，所以不论是佛教还是基督教的初始教义中都明确禁止偶像崇拜。

8月28日

要勇敢的和过去告别，牛顿第三定律告诉我们，想要前进总得留下点什么。

\
8月30日

很多事只有做了才能理解，不做是无法理解的，这大概是我觉得对「知行合一」最好的解释。人的某些认知是需要通过行动提高的，因为人是无法通过空想去想象得到某种东西后的体悟，就像祛魅某种事物的最好办法是拥有它。



8月31日

我们穷其一生到底在追求什么？社会教育我们说是钱权色，但其实我们其实一生都在追求美好的生活、美好事物、美好的人际关系、美好的体验。而要体验美好的东西往往是稀缺性的，是需要资源的匹配和价值交换的，这一切都是有代价的。人们追求金钱等一般等价物的初衷永远都是为了体验美好事物，但它是只是一种体验，而不能永远拥有。



很多事，只要你认真你就能超过三成的人。这个世界的草台班子化比我们预想的要严重，很多人连基本的认真都做不到，所以不论你做的事有多难，你只要认真就能轻轻松松超过30%的人，再努努力再拿下30%就能及格。所以即便是再难的事也不要怕。只要是人做出来的，一切都有迹可循。



9月1日

如果没有参与权，就很难激发他人的利他心。当人无法决定任何东西的时候，人会优先选择利己。



9月5日

有时候你看一个人推崇的书籍、音乐和电影就能反应其价值取向。文化是上层建筑，而文化最直接的表现就在文艺作品和严肃文学的喜好上。



9月7日

沉溺于文学和摇滚，如同沉迷于无休止的纵欲。



9月9日

过去是一条回不去的路，我们始终要往前走。我以前特别爱追忆往事，现在别人一提过去多好多好，我就觉得可笑。场域里是不存在永恒秩序的，唯一的不变是永远会改变。追悔莫及不如积极拥抱变化，务实奋进。



9月11日

我喜欢和迷恋文字，所以我还是喜欢做一个只能发文字的产品。在充斥着图片和视频的信息流里文字有天然的劣势。



9月14日

当一个人追求幸福的权利被剥夺时，别人的痛苦就成了他的快乐。



9月15日

用户需要的是花哨的功能吗？不，用户需要的是有趣的内容。



9月16日

鲁迅先生说过，我们极容易变成奴隶，而且变了后会万分欢喜。每个人可以有选择自己如何生活的权利，但请别在别人为你争取自由的时候说人家你怎么不知趣。有没有可能不是每个人都像你这样想当奴隶呢？

\
9月18日

人的心情似乎不是根据当下来显现的，而是关乎未来，对于未来的预期决定了此刻的心情与心态。因为有的人心怀希望，而有的人只剩下回忆。

\
9月21日

美就意味着真实，虽说真实的东西不一定都美，可是最美的东西永远是真实的。丧失了真实性，就丧失了艺术性。

这个世界没有什么是免费的，好的东西都需要付出耐心。



9月22日

人生谜底，寻寻觅觅。每一次和老友相逢又别离总是有一种难以言说的情绪。千言万语也道不尽往昔与未来，每次遇见也都能感受到岁月在我们彼此身体上和灵魂里留下了无法掩盖的痕迹。虽说我们不可避免的接受了生命的暗淡，生活也都让我们变得越来越规律和平凡。《诗经》里说“靡不有初，鲜克有终”。但我总奢侈的希望我的朋友们都能寻找到属于自己人生的答案，希望朋友们也都过上理想的生活。



9月23日

为什么每次坐卧铺都能遇到鼾声很大的人，这或许就是墨菲定律吧。抑或者说打呼噜的人口比例已相当的高了。



9月24日

敏感体质+丰富的表达欲，才是一个内容创作者最大的福报。充沛情绪加上精准的用词，真的是老天爷追着赏饭。但也别忘了，天予不取，反受其咎。



有趣这个词是特别有意思的，它简直是衡量人类之间是否是同类的重要标准。一个“有趣”的人不会认为无聊的人有趣，同样一个“无聊”的人也不会认为有趣的人有趣。也就是说无聊的人从不会自诩无聊，而有趣的人会认为有趣的人比他更有趣。



生命的本质是体验，所以越来越理解高更了，追逐梦想就如同追逐自己的噩运。



9月25日

在一个生产力过剩和刚需被挖掘殆尽的时代，仅仅凭借着掌握生产资料已经不能单纯的造富了。要么去做新兴产业，要么结合目前的资源去挖掘更多的细分场景。总之，增长才是决定未来生存的关键。

有时候你越想要你反而得不到，你释然了，它反而给你了。命运的无常捉弄是人生一大乐趣之一，可能人类对于上帝来说就是蛐蛐，上帝偏爱逗蛐蛐。



9月26日

现在社会的一大矛盾就是虽然一些人识字了，但其实有严重的阅读障碍。他们很难看懂文字所表达的真情流露。而是以一种秀无知的状态在贬损一些有趣的思考，但因这个群体越来越大，这种情绪反而成了主流。



9月29日

人的使命感很重要，虽然有使命感并不一定能做成什么事情。但是有使命感是出发做一件事的基础。做足万全准备的事儿是少有的，要敢于在什么都缺少的情况下即便弄脏双手也要出发。知行合一，很多事不做是无法理解的。
