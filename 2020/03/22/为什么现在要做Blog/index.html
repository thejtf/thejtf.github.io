为什么现在要做Blog

2020-03-22#Blog

它是一种反抗，对于如今信息孤岛的反抗。

前言

搞了几天的Blog终于算是确定下来了，这篇文章就说明一下为什么要做这个站点吧。我想没有一件事是毫无道理的，内心想做一件事的原因一定是积攒多年的爆发。

或许有人会有疑问为啥现在还要自己在建一个博客，而不是利用微信公众号、微博等自媒体平台呢？

非自媒体

算了算从一开始搞公众号到现在也已经6年有余了，现在越来越觉得把一些内容放到公众号里不够踏实，我曾经以为这公众号就是我的，可是越来越发现它并不是我的，一些由于你我都懂的原因，蚕食那些少的可怜的表达欲。

和主流自媒体产生的内容不同，个人博客一般是自己的记录和分享以人为单位的。它不必跟热点，它更多的应该是自我的表达，当然这样带来的就是内容对于读者的喜好来讲参差不齐，而不是像算法投喂的那样，更多的是读者感兴趣的内容。所以在当下，它某种程度上是一种反抗。

我想，我们这代人每个人都是开放互联网的受益者，每天都从网上获取别人分享的知识和经验，久而久之，我也想跟着前辈一样做一个属于自己的地方，记录我想记录的全部，而不是为了流量去大声的呐喊，更不应是发完图文后焦虑的关注阅读量。

它更应该像一座宝藏，属于认可它的宝藏。

GitHub

以前我做Blog都是搭建在自己买的阿里云上，但是这次我选择部署在GitHub上的，虽然说github在国内访问不是很稳定，但毕竟这个Blog除了我自己也没几个人访问。这样做可以省去很多麻烦，Git也能管理程序的版本，方便做一些二次开发。

尾巴

搭建一个属于自己的博客会培养自己的编程兴趣，比如现在我就更了解Git了，由于是静态博客，GitHub上的贡献格子也可以更好的记录我的博客更新记录。

人生似乎是一场不断给自己挖坑的过程，我们也在不断的填坑的过程中锻炼和成就了自我，或许这就是生命的乐趣吧。
