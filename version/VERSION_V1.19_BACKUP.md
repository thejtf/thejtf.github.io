## 🆕 V1.19 更新内容

### 新增功能

#### 1. 书籍分类系统

将读书笔记分类改为5大标准分类：

| 分类 | 包含类型 |
|------|----------|
| **文学** | 小说、诗歌、散文、戏剧、童话、古典文学 |
| **社科** | 历史、哲学、政治、经济、法律、社会学、心理学 |
| **科技** | 数理化、生物、医学、计算机、工程、科普 |
| **艺术** | 绘画、音乐、摄影、设计、建筑、影视 |
| **实用** | 生活、健身、烹饪、育儿、理财、旅行、工具书 |

#### 2. 书籍标签系统

新增书籍阅读价值标签：

| 标签 | 说明 |
|------|------|
| **经典** | 值得反复读的传世之作 |
| **必读** | 对自己很重要的书 |
| **泛读** | 随便翻翻即可（默认） |
| **参考** | 工具书、资料类，按需查阅 |

#### 3. 分类和标签映射表

在 `scripts/kindle-sync.js` 中添加映射表：

```javascript
const BOOK_CATEGORIES = {
  '百年孤独': '文学',
  '中国历代政治得失': '社科',
  // ...
};

const BOOK_TAGS = {
  '百年孤独': '经典',
  '中国历代政治得失': '必读',
  // ...
};
```

### 修改文件

```
scripts/kindle-sync.js               # 添加分类和标签映射逻辑
source/_reads/*.md                   # 更新所有文章的分类和标签
```

## 🔄 备份恢复方式

### 恢复步骤

1. **克隆项目**
```bash
git clone -b source git@github.com:thejtf/thejtf.github.io.git
cd thejtf.github.io
npm install
```

2. **启动本地预览**
```bash
npm run server
# 访问 http://localhost:4000/
```

3. **验证分类和标签**
- 访问 http://localhost:4000/read/
- 侧边栏应显示：文学、社科、实用（分类）
- 标签显示：经典、必读、泛读

4. **部署**
```bash
npm run deploy
```

### 关键文件说明

| 文件 | 用途 |
|------|------|
| `scripts/kindle-sync.js` | 包含 `BOOK_CATEGORIES` 和 `BOOK_TAGS` 映射表 |
| `source/_reads/*.md` | 读书笔记文章 |

### 添加新书分类和标签

编辑 `scripts/kindle-sync.js`：

```javascript
const BOOK_CATEGORIES = {
  '新书名': '文学',  // 或 社科/科技/艺术/实用
};

const BOOK_TAGS = {
  '新书名': '经典',  // 或 必读/泛读/参考
};
```
